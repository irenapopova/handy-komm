### Full Stack Application

## Description

Full stack app using RESTful API design.

- Responsive front-end design done with Material-UI,
- Passport used for authentication, database hosted on MongoDB.

## Technologies & Tools

### Front-end:

- React
- Redux
- Redux-Saga
- Material-UI
- Webpack
- Axios

### Backend:

- Node/Express
- MongoDB/Mongoose
- Passport
